
# Installation

Clone the repository and do:

    npm install

# Building the library

    npm run build
    
## Install the SystemJs Example 
    
    cd examples/systemjs
    npm install
    npm run build
    npm start
    
Then access [http://localhost:8080](http://localhost:8080)

## Install the Webpack Example 
    
    cd examples/webpack
    npm install
    npm start
    
Then access [http://localhost:8080](http://localhost:8080)
